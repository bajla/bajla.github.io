Our Winning Projects

  • Cryptography with Eliptic Curves: We researched the mathematical principles behind elliptic curve cryptography and developed a library in c# based on our findings in which we also implemented diffie-hellman key exchange. The outcome was the creation of a user application that demonstrates the implementation of the obtained knowledge and allows users to engage in encrypted two-way communication through a P2P network. In addition, we compared the performance of our library with a standard encryption library provided by Microsoft and evaluated the resistance of cryptography against attacks from quantum computers. This project earned us third place in the national competition. Achieved by Aney Vaishnav and Maj Zabukovnik.
    link to the research paper.
  • Blockchain for News Publication:We built our own blockchain from scratch using the library mentioned above. The research project aimed to explore blockchain technology and create our own implementation. With over 10,000 cryptocurrencies worldwide, we focused on a niche that supports both monetary transactions and news publication, ensuring integrity and preventing censorship. We used strong cryptographic algorithms and implemented a reward system for network stability and self-sufficiency. Additionally, we developed a user-friendly web application for viewing and contributing news. This project also secured third place in the national competition. Achieved by Aney Vaishnav and Maj Zabukovnik.
    link to the research paper.
  • Connecting articial intelligence with a virtual avatar: This research paper is aimed at a possible connection between professors and AI. We have built a website that serves this purpose. The frontend is built using NextJS and uses AuthJS to authenticate users. All of the data is stored in MongoDB and the data is being served using API built in NestJS. The API also serves as a bridge between the frontend (Handling user input) and backend (Uploading that input to OpenAI API and returning a voice file from Elevenlabs API). This got us a bronze award. Achieved by Miha Kos.
